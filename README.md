## Terminal Commands

Build and run the program.

```sh
make && ./a.out
```

Clean out build files.

```sh
make clean
```

Build and run testing program.

```sh
make DRIVER=src/test_driver.cpp && ./a.out
```

Clean out test build files.

```sh
make DRIVER=src/test_driver.cpp clean
```
